base.BaseDialogFragment

描述：用于实现Dialog内的Fragment，此Fragment不可被贴到Activity或是主窗口的Detail部分

* 方法

void initComponents();
初始化组件

void initEvents();
初始化事件监听

void initLogic();
初始化业务逻辑

int getFragmentLayoutResId();
必须返回Fragment加载的layout文件

* 事件

无

* 内部对象

View innerView
内部视图对象，该对象可以使用findViewById()来查找layout文件内的组件
