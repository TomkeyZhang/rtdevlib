base.BaseMainActivity

描述：用于实现主界面，该界面在横竖屏情况下，分别将显示两列和一列，其左侧是索引，右侧是明细，程序内必须有一个主界面，且只能有一个

* 方法

void initOnce();
启动时仅初始化一次的代码段

String getBarTitle();
必须返回主界面的标题

Fragment getFragment(int currentFragment);
替换明细的Fragment，该函数在横竖屏切换时被触发，跟据GlobalInstance内的currentFragment参数进行索引

Fragment getIndexFragment();
替换索引的Fragment

void initMenu(Menu menu);
初始化菜单

void loadFragments();
加载所有的Fragment

void releaseFragments();
释放所有的Fragment

* 事件

void onHomeClick();
Home键被按下时触发

void onRecentAppClick();
RecentApp键被按下时触发

* 内部对象

boolean oneTimeRun
获取主界面内，initOnce()是否已经执行，或是否需要再次执行
