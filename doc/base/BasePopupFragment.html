base.BasePopupFragment

描述：用于实现Fragment，此Fragment可以被贴到继承自BasePopupActivity的明细部分，并按横竖屏模式自动更改布局

* 方法

void initComponents();
初始化组件

void initEvents();
初始化事件监听

void initLogic();
初始化业务逻辑

int getFragmentLayoutResId();
必须返回Fragment加载的layout文件

int getBarTitle();
必须返回Fragment的标题，不包含路径

int getBarTitleWithPath();
必须返回Fragment的标题，包含路径

String getMainActivityName();
必须返回主界面的类名

void initMenu(Menu menu);
初始化菜单

void setNewArguments(Bundle bn); 为Fragment重设参数

* 事件

void onGetNewArguments(Bundle bn);
Fragment获得新的参数时触发，参数以Bundle形式传入

* 内部对象

View innerView
内部视图对象，该对象可以使用findViewById()来查找layout文件内的组件

Bundle innerBundle
内部Bundle对象，在为Fragment重设参数时，该对象被填充
