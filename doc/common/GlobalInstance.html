common.GlobalInstance

描述：内部全局对象类

* 方法

void init(Context context, boolean debug);
初始化全局对象，在程序启动时必须且仅能调用一次

* 事件

无

* 内部对象

boolean DEBUG
是否调试模式

boolean dualPane
是否双屏模式，该参数的值由屏幕朝向自动决定

PackageManager pm
内部包管理器

int currentFragment
当前/最后所展示的Fragment序号，需要在切换Fragment时进行设置，并在BaseMainActivity的getFragment()方法内通过序号来获取Fragment

boolean loadingNetwork
是否正在加载网络状态

NetworkInfo networkInfo
当前的联网状态

String networkSpeed
当前的网速，以字符串表示

String nameServer
用于nslookup的DNS服务器，默认为8.8.8.8(Google DNS)

DeviceInfo device
当前设备的信息
