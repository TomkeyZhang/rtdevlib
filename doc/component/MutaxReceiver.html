component.MutaxReceiver

描述：用于实现互斥的消息接收器

* 方法

MutaxReceiver(String main, String progress, String[] mutax);
构造函数，必须传入主Action，过程Action，互斥Actions，其含义在于，当主Action在执行时，挂起所有与之互斥的Actions，另外，在主Action执行时，可以同时以过程Action的形式抛出执行进度。

void setOnReceiveMessage(OnReceiveMessage onReceive);
设置接收到消息时的监听器

void register(Context context);
向系统注册Receiver

void unregister(Context context);
反注册Receiver

* 事件

void onStateChange(boolean operating);
状态变更时触发，状态即是Service是否在执行中

void onProgress(String name, int position, int total);
执行过程变更时触发，传回的参数为正在执行的内容，当前执行的任务序号，总共需执行的任务数

void onMutaxMessage(boolean operating);
互斥消息发生时触发，将挂起所有互斥的Actions

* 内部对象

无
