utils.DownloadUtils

描述：用于实现下载，以及下载后的动态填充功能

* 方法

void downloadFileT(final Context context, final ImageView iv, final String url, String localDir, final String localFile, final Handler hProgress);
实现异步下载文件，若是在下载时提供了ImageView，则下载后自动将文件视为图片并填入ImageView中，Handler用于返回下载的进度

* 事件

* 内部对象

int WHAT_DOWNLOAD_START;
开始下载时抛出此消息，消息中arg1为下载进度，arg2为文件大小

int WHAT_DOWNLOAD_PROGRESS;
下载过程中抛出此消息，消息中arg1为下载进度，arg2为文件大小

int WHAT_DOWNLOAD_FINISH;
下载结束后抛出此消息，消息中arg1为0，arg2为文件大小
