utils.ImageUtils

描述：用于实现通用的图形图象操作

* 方法

Bitmap getBitmapByAssets(Context context, String path);
从assets内读取一张图片

Bitmap roundedCornerBitmap(Bitmap bitmap, float radis);
将图片转换成圆角图片，radis将决定圆角的大小

Bitmap blackWhiteBmp(Bitmap bmp);
将图片转换成黑白图片

Bitmap rotateBmp(Bitmap bmp, int angle);
旋转图片，angle决定了旋转的角度

Bitmap flipBmp(Bitmap bmp, int mode);
翻转图片，mode决定了翻转的方向，1为横向翻转，2为纵向翻转

Bitmap matrixBmp(Bitmap bmp, int margin);
压缩图片到正好适应屏幕，margin为左右边距

byte[] readFileImage(String filename);
读入图片文件到byte[]

saveBitmapToFile(Bitmap bmp, String fileName, CompressFormat format);
将内存中的图片保存到文件

Bitmap colorMatrixBmp(Bitmap bmp, float[] matrixSrc);
对图片作色矩覆盖，matrixSrc是一个5x4的色值矩阵
