utils.NetworkUtils

描述：用于实现通用的网络状态操作

* 方法

void doGetNetworkInfoT(final Context context);
异步获取网络信息，在获取时，GlobalInstance.loadingNetwork将被置为true，获取完毕时，GlobalInstance.loadingNetwork被置为false。
在获取完毕后，可以通过GlobalInstance.networkInfo来获取当前的网络状态，可以通过GlobalInstance.networkSpeed来获取当前的网速

String getNetworkStatusString(final Context context);
将当前网络信息组装成一个字符串返回
